# Test Sandbox

This folder is a small compiler sandbox used to inspect preprocessing, assembly, and object output alongside the source.

## Files

- [main.cpp](main.cpp): the source file for the sandbox example.
- [main.i](main.i): preprocessed output generated from the source.
- [main.s](main.s): assembly output generated from the source.
- [main.o](main.o): object file produced during compilation.
- [main](main): local executable build artifact.

## Notes

- These files are kept here for inspection and learning.
- Generated outputs are ignored by the repository-level `.gitignore`.
