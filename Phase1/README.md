# Phase 1 Exercises

This folder groups the first set of standalone C++ practice programs. Each source file matches one concept, and the matching executable in the same folder is treated as a local build artifact.

## Topics

- Arithmetic - [arth.cpp](arth.cpp) - `arth`
- Conditions - [conds.cpp](conds.cpp) - `cond`
- Constants - [constexpr.cpp](constexpr.cpp) - `const`
- Type conversion - [conversion.cpp](conversion.cpp) - `conv`
- do-while loop - [doWHile.cpp](doWHile.cpp) - `dowhile`
- for loop - [for.cpp](for.cpp) - `for`
- Input handling - [inp.cpp](inp.cpp) - `inp`
- Operators - [opers.cpp](opers.cpp) - `opers`
- sizeof - [sizeof.cpp](sizeof.cpp) - `sizeof`
- switch statement - [swt.cpp](swt.cpp) - `swt`
- while loop - [while.cpp](while.cpp) - `while`

## Notes

- The executable names mirror the source topics.
- Generated outputs are ignored by the repository-level `.gitignore`.
